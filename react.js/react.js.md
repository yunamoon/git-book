# React.js

### 1. React

> 메타에서 개발한 오픈소스 자바스크립트 라이브러리.
>
> 대규모 웹 서비스의 UI를 더 편하게 개발하기 위해 만들어진 기술이다.

### 2. React의 기술적 특징

* 컴포넌트를 기반으로 UI를 표현한다.
* 화면 업데이트 구현이 쉽다.
* 화면 업데이트가 빠르게 처리된다,
