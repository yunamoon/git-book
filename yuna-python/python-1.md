# Python 기초 문법

### 파이썬 문자와 변수

```python

# 출력
print(123)
print('문유나')
print("문유나")

# 연산
print(123+456)
print(100*100)

# 변수
name = "문유나"
print(name)

# 인덱스
name = "가나다라마바사"
print(name[0])
print(name[0:3])
 
```
